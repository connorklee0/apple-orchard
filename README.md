# Macbook Landing Page

A modern e-commerce website inspired by Apple's iconic design aesthetic, showcasing premium tech products with immersive 3D visualizations and cinematic scroll animations.

> **Note:** This project was built following a tutorial to learn advanced frontend techniques including 3D rendering, scroll-based animations, and performance optimization. The design and user experience are inspired by Apple's website.

## Features

- **3D Product Visualization** - Interactive Three.js scenes showcasing products from multiple angles
- **Scroll-Driven Animations** - Smooth GSAP animations triggered by scroll position for cinematic effects
- **Pinned Sections** - Dynamic content sections that pin and animate during scroll
- **Responsive Design** - Fully responsive layouts optimized for all devices using Tailwind CSS
- **Performance Optimized** - Lazy loading and efficient rendering for smooth 60fps experiences

## Built With

- React.js
- Three.js (3D rendering)
- GSAP (GreenSock Animation Platform)
- Tailwind CSS
- Vite
